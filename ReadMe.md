# Blackjack Game (C# .NET Core Console App)
By: Siphesihle Dube

## Features
+ Single-player
+ Player can Hit
+ Player can Stand
+ Player can Double
+ Player can Surrender

## To Play
1. Run the program.
2. Enter your bet, minimum bet of 10 out of 500 chips).
3. Play your hand. 
    * Select actions with words (e.g. hit, stand etc).
4. Play continues until you run out of chips.
